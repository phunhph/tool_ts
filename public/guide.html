Hướng dẫn tạo file câu hỏi Excel

Để tạo đề thi trắc nghiệm, bạn cần chuẩn bị một file Excel (.xlsx hoặc .xls) theo đúng định dạng dưới đây.

1. Cấu trúc file Excel

File Excel cần có các cột theo thứ tự sau:

Tên cột Mô tả Ví dụ
Câu hỏi Nội dung câu hỏi Thủ đô của Việt Nam là gì?
Đáp án 1 Lựa chọn A Hà Nội
Đáp án 2 Lựa chọn B Hồ Chí Minh
Đáp án 3 Lựa chọn C Đà Nẵng
Đáp án 4 Lựa chọn D Hải Phòng
Phương án đúng Đáp án chính xác (Phải trùng khớp hoàn toàn với một trong các lựa chọn) Hà Nội
